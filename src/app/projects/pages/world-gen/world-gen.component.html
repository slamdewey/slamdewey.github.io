Procedural World Generation

Overview

This is an interactive walkthrough of a procedural world generation system that uses weather simulation techniques to produce realistic biome distributions. The system builds a world through sequential stages: terrain generation, climate modeling, and biome classification.

Each stage below shows a visualization of the generated data. Adjust the parameters and click Regenerate to see how they affect the output.

Stage 1: Fault Lines

The generation begins by simulating tectonic plate boundaries. Two noise sources create continent-scale features (plate noise) and mountain ridge boundaries (ridge noise). The ridge noise uses the formula (1 - |noise|)^3 to create sharp ridges from smooth noise. Both sources use cylindrical coordinate wrapping via 3D noise evaluation, so the world tiles seamlessly horizontally.

Stage 2: Elevation

Elevation is generated using Fractal Brownian Motion (fBm) — multiple octaves of OpenSimplex noise layered at increasing frequencies and decreasing amplitudes. The raw noise is blended 50/50 with the fault line data to create terrain that follows tectonic boundaries. Sea level is determined by the water percentage parameter: all elevation values are sorted and the percentile cutoff becomes sea level.

Stage 3: Temperature

Temperature is primarily driven by latitude using an inverse parabola function T(y) = 1 - (2y - 1)^2, which gives maximum warmth at the equator and minimum at the poles. A small noise jitter adds local variation. Above sea level, an atmospheric lapse rate reduces temperature with elevation (higher = colder). Ocean cells are also cooled by a fixed factor.

Stage 4: Wind

Global wind patterns are modeled after Earth's atmospheric circulation cells. Six latitudinal zones (three per hemisphere) represent the Hadley, Ferrel, and Polar cells. Each zone has alternating vertical and horizontal dominant wind directions, with smooth interpolation between zones and noise-based jitter for natural variation.

Stage 5: Precipitation

This is the most complex stage — a multi-cycle weather simulation. Each cycle: ocean cells evaporate moisture into clouds; clouds are pushed by wind (advection); precipitation falls based on temperature (warmer air holds more moisture); excess moisture runs off to lower-elevation neighbors. After several cycles, realistic moisture patterns emerge — rain shadows behind mountains, wet coastlines, and dry interiors.

Stage 6: Biome Classification

Finally, each cell is classified into a biome using a Whittaker-diagram-inspired 6×6 lookup table indexed by moisture and temperature. Water cells are classified by depth (shallow vs. deep ocean), and cells above a high elevation threshold become mountains. The remaining land is mapped to one of nine terrestrial biomes: Arctic, Tundra, Desert, Grassland, Savanna, Boreal Forest, Temperate Forest, or Rainforest.
